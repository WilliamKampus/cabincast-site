サポート

CabinCast の問題の多くは、ホットスポット、正確なビューア URL、 選択中のモード、または高品質ストリーミング中の iPhone の発熱に関係します。

ビューア URL http://240.3.3.3:8080

接続チェック

  • Tesla が iPhone のパーソナルホットスポットに接続されていることを確認します。
  • https:// ではなく http://240.3.3.3:8080 を使います。
  • iOS のローカルネットワークまたは VPN の確認が表示された場合は許可します。
  • ビューアは読み込まれるのに映像が動かない場合は、CabinCast のブロードキャストを再開始します。

モード確認

  • 新しい車両や iPhone で設定する場合は、まず Stable Mode を使います。
  • Stable Mode が安定して動いてから Quality Mode に切り替えます。
  • サポーター向けの PRO Codec、High Quality、Smooth Mode は、ホットスポット接続と iPhone の発熱状態に左右されます。
  • 発熱警告が出る場合は、品質を下げてから再度テストしてください。

フルスクリーン

  • True fullscreen は Tesla ブラウザで使えるサポーター機能です。
  • フルスクリーンを開く前に、明るい画面についての警告が表示される場合があります。
  • フルスクリーン後にストリームが止まる場合は、ビューアに戻って Reset Stream を使います。

既知の制限

  • CabinCast は表示専用で、Tesla 画面から iPhone アプリを操作することはできません。
  • DRM 保護された動画は、iOS の画面収録制限により黒く表示される場合があります。
  • Tesla ブラウザは停車中かつ地域の法令に従って使用してください。
  • 音声は Bluetooth または iPhone から通常どおり再生してください。

メールに含める情報

iPhone の機種、iOS バージョン、Tesla ソフトウェアバージョン、 選択した CabinCast モード、ビューア URL を開いた後に起きたことを送ってください。
