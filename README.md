# CabinCast Website

This repository is the public static site intended for GitHub Pages.

- Custom domain: `cabincast.wirurun.com`
- Support URL: `https://cabincast.wirurun.com/support.html`
- Privacy URL: `https://cabincast.wirurun.com/privacy.html`

Recommended GitHub Pages source:

1. Branch: `main`
2. Folder: repository root
3. DNS: add a `CNAME` record for `cabincast.wirurun.com` pointing to the GitHub
   Pages host for this repository.
