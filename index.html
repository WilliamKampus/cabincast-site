Basic setup

Use CabinCast only when it is safe and legal to use the vehicle browser.

1

Start casting on iPhone

Open CabinCast, start the local connection, and begin the iOS screen broadcast.

2

Connect the Tesla

Connect the Tesla to the iPhone Personal Hotspot.

3

Open the viewer

Open http://240.3.3.3:8080 in the Tesla browser.

Designed around Tesla browser limits

CabinCast includes multiple viewer modes for different conditions: WebRTC for the normal path, software decode fallback paths for compatibility, and diagnostics when the connection needs troubleshooting.

View-only

CabinCast shows the iPhone screen. It does not control iPhone apps from the vehicle screen.
